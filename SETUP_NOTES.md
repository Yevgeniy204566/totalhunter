# BattleBot — Заметки по настройке

## Профили калибровки

Бот поддерживает 3 профиля: **Client**, **Chrome**, **Firefox**.
Каждый профиль хранится в `profiles/profile_*.json`.

---

## Профиль Client (игровой клиент)

Запускай игру через официальный клиент.
- Офсет: **0** (по умолчанию)
- Калибровка нужна один раз, сохраняется кнопкой «Сохранить»

---

## Профиль Chrome / Firefox (браузер в оконном режиме)

Браузер в обычном окне (не полный экран).

**Особенность:** игра в браузере смещает диалоговые окна (список склепов, окно ускорения) чуть ниже, чем в клиенте — из-за панели вкладок и адресной строки браузера.

**Что делать:**
1. Вкладка «СКЛЕПЫ» → выбрать профиль **Chrome**
2. Запустить бота, посмотреть куда кликает
3. Если промахивается по кнопкам — подобрать **Смещение ↑↓**
   - Типичное значение: **+45...+55 px**
4. Нажать **💾** — сохранится в профиль

> **Заметка:** если браузер открыт на весь экран (F11), игра занимает всё пространство и офсет не нужен — работает как клиент (значение 0).

---

## Вывод по офсету

| Режим запуска | Смещение ↑↓ |
|---|---|
| Клиент | 0 |
| Браузер (оконный режим) | ~+50 (подобрать) |
| Браузер (F11, полный экран) | 0 |

---

## Где настроить офсет

Вкладка **«СКЛЕПЫ»** → строка настроек:

```
Профиль: [Chrome ▼]   Смещение ↑↓: [−][45][+]   [💾]
```

- **[−] / [+]** — изменить на 5 px за шаг
- **[💾]** — сохранить в выбранный профиль

---

## Калибровка якорных точек

Вкладка **«КАЛИБРОВКА»** → кнопка «Калибровать».

Программа покажет две точки поочерёдно:
1. **Точка А** — центр мини-карты (нижний левый угол экрана)
2. **Точка Б** — иконка серебра (верхний правый угол экрана)

После калибровки нажать **«Сохранить»** → профиль обновится.

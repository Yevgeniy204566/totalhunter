# Дизайн модуля Склепов — `crypt_hunter.py`

**Дата:** 2026-04-05  
**Статус:** Одобрен  

---

## 1. Цель

Автоматический сбор склепов в игре Total Battle. Пользователь выбирает типы склепов, задаёт количество ускорений и запускает бот. Бот работает без участия игрока: ищет склепы через меню Дозорной башни, отправляет Капитана Картера, ускоряет марш, ждёт возвращения и повторяет цикл.

---

## 2. Файлы

| Файл | Роль |
|---|---|
| `crypt_hunter.py` | Весь игровой цикл — класс `CryptHunter` |
| `main.py` | Добавляет вкладку «Склепы», вызывает `start()/stop()` |
| `targets/crypts.pt` | YOLO-модель для детекции склепов |
| `targets/*.png` | Иконки 31 типа склепов для GUI |

---

## 3. Типы склепов (31 штука)

- **Ordinary 1–12** — обычные (12 типов)
- **Epic 2–18** — эпические (17 типов)
- **R 1–2** — редкие (2 типа)

Иконки из `C:\BattleBot\targets\*.png` — отображаются в GUI без подписей.

## 4. Полный цикл бота

```
[1] Открыть Дозорную башню (клик по иконке)
[2] Кликнуть «Склепы и арены» в боковом меню
[3] Скроллинг вниз + YOLO режим 1
    ├── нашёл нужный тип → клик «Перейти» → [5]
    └── конец списка → [4]
[4] СБРОС: клик «Арена» → пауза ~0.6 сек → клик «Арена» снова
    (два одиночных клика — открыть и закрыть → сброс поиска на ноль)
    + рандомная пауза 0.4–0.9 сек → вернуться к [3]
[5] Игра телепортирует на карту
[6] YOLO режим 2: найти склеп в центре экрана → кликнуть
    └── не нашёл → РЕСТАРТ с [1]
[7] Диалог склепа открылся
[8] Читаем масло (OCR зоны выше кнопки «Исследовать»):
    - Берём только первую цифру до «/»
    - Парсим: «6,5м» → 6_500_000 | «74,7к» → 74_700 | «1,06м» → 1_060_000
    - Если < 70_000 → ПОЛНЫЙ СТОП (нет масла, нужен игрок)
[9] Клик «Исследовать»
    └── кнопка не реагирует → склеп захвачен → РЕСТАРТ с [1]
[10] Клик по полосе события Картера (вверху, чуть правее центра)
[11] УСКОРЕНИЕ: до N кликов «Использовать» (N = настройка 1–5)
     Правило: если оставшееся время ≤ 15 сек — прекращаем ускорять
     После каждого ускорения: remaining_time /= 2
[12] OCR: читаем оставшееся время из диалога ускорения
     wait_time = remaining_time × 2 + 2 сек (запас)
[13] Клик по игровому полю (закрыть диалог)
     spend_credit() — списать 1 кредит
[14] sleep(wait_time)
[15] → вернуться к [1]
```

---

## 5. Аварийные остановки

| Причина | Действие |
|---|---|
| Масло < 70к (OCR шкалы) | ПОЛНЫЙ СТОП — статус «Нет масла», ждём игрока |
| «Исследовать» мертва (масло OK) | РЕСТАРТ — склеп захвачен другим игроком |
| YOLO не нашёл склеп на карте | РЕСТАРТ с [1] |
| ESC | ПОЛНЫЙ СТОП (глобальный хук в main.py) |

## 6. Класс `CryptHunter`

```python
class CryptHunter:
    def __init__(self)
        # Загружает targets/crypts.pt единожды

    def start(
        selected_crypts: list[str],   # ["Ordinary_1", "Epic_5", ...]
        conf: float,                   # точность YOLO
        accelerations: int,            # сколько раз ускорять (1–5)
        on_found_callback,             # GUI: найден склеп
        on_status_callback,            # GUI: обновить статус
        on_stop_callback,              # GUI: аварийная остановка
    )

    def stop()

    # Приватные:
    _open_watchtower()
    _select_crypts_tab()
    _scroll_and_find(selected: list[str]) -> str | None
    _reset_search()                    # клик Арена × 2 с паузой 0.6с
    _travel_to_crypt()
    _detect_on_map() -> bool           # YOLO режим 2
    _send_captain() -> bool            # False = кнопка мертва
    _read_oil() -> float               # OCR → число (абсолютное)
    _accelerate(n: int) -> float       # возвращает remaining_time (сек)
    _read_remaining_time() -> float    # OCR диалога ускорения
    _close_dialog()                    # клик по игровому полю
    _random_pause()                    # sleep(0.4–0.9)
    _click(x, y, jitter=6)            # клик со смещением 5–8 px
```

## 8. OCR масла — формат парсинга

```python
# Зона OCR: выше кнопки «Исследовать» в диалоге склепа
# Берём только до «/» (остальное — вместимость Картера, не нужно)
# Примеры входных строк:
#   «6,5м/74,7к»  → берём «6,5м»  → 6_500_000
#   «74,7к/74,7к» → берём «74,7к» → 74_700
#   «1,06м»       → берём «1,06м» → 1_060_000
# Порог остановки: < 70_000
```

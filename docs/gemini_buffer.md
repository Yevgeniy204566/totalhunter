Протокол «Total Hunter» активен. Памятку по системе бэкапов зафиксировал. 

### 🛡 Резюме системы бэкапов (GCP 34.68.86.57)

* **Объект:** PostgreSQL (`users`, `transactions`, `orders`, `referrals` и др.).
* **Режим:** Автоматический (Cron), каждые 6 часов.
* **Ротация:** Хранение 28 последних копий (глубина 7 дней).
* **Локация:** `~/backups/` на сервере.

### 💻 Команды для терминала (Активация)

Выполни это на сервере, чтобы запустить систему:

```bash
# Деплой и права
cd ~/total-hunter-backend && git pull
chmod +x scripts/backup_db.sh

# Тестовый прогон (проверка создания файла)
bash scripts/backup_db.sh && ls -lh ~/backups/

# Установка в планировщик (Cron)
(crontab -l 2>/dev/null; echo "0 */6 * * * /home/ubuntu/total-hunter-backend/scripts/backup_db.sh >> /home/ubuntu/backups/backup.log 2>&1") | crontab -
```

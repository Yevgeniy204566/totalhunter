### Один мелкий нюанс на будущее (по поводу Git)

Судя по логам, у тебя возникла заминка со `git stash` из-за файлов внутри папки `__pycache__`:

> `error: Your local changes to the following files would be overwritten by merge: __pycache__/coord_manager.cpython-313.pyc`

То, что скомпилированные файлы `.pyc` мешают операциям Git — это классический признак того, что папка кэша Python случайно попала в индекс репозитория (была добавлена через `git add .` до того, как настроили игнорирование).

Чтобы каждый раз не прописывать `git checkout -- __pycache__/`, этот мусор стоит раз и навсегда убрать из отслеживания Git. Сделать это можно двумя простыми шагами:

1. Добавь (если еще нет) в файл `.gitignore` в корне проекта строчки:
```text
__pycache__/
*.pyc

```


2. Выполни команду в консоли, чтобы удалить уже проиндексированный кэш из репозитория (сами файлы с диска не удалятся, Git просто перестанет их замечать):
```bash
git rm -r --cached **/__pycache__/ *.pyc

```

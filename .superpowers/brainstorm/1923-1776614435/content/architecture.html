Как связаны Vercel, GCP и Google OAuth

Поток авторизации
🌐 Vercel — total-hunter.vercel.app
React SPA, Google OAuth кнопка
↕ ID Token (Google)
🔑 Google OAuth 2.0
Верифицирует пользователя → возвращает ID Token
↕ ID Token → JWT
⚡ FastAPI GCP :8000
Проверяет токен, выдаёт JWT, хранит сессию
Привязка HWID (код из бота)
1 Бот генерирует 6-значный код → отправляет на сервер вместе с HWID
2 Код живёт 10 минут в БД (link_codes таблица)
3 Пользователь вводит код на сайте (залогинен через Google)
4 Сервер пишет HWID в запись пользователя → аккаунты связаны ✅
